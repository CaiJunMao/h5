语句（if条件语句，switch条件语句）和程序结构

			语句：
				语句通常使用一或多个关键字来完成给定任务，
				我们写的任何一行代码其实都是一条语句。一个程序总是由很多的语句来组成
			结构化程序有三种结构。顺序结构、选择结构、循环结构
			if条件语句：
				if(){}
				if…else
				if (){}else if(){}…else if(){}else{}
				注：只执行一个分支
			switch条件语句：
				switch (表达式) {
				  case value1: 
				    //语句1
				  break;
				  case value2: 
				    //语句2
				  break;
				  ……
				  default: 
				    //default语句
				    break;
				}
				表达式可以是任意，遇到break才会返回，否则每个case中的语句都会被执行
