语法基本要求

			JavaScript严格区分大小写，大小写敏感 
			JavaScript使用 ;  结束语句。分号  ; 可以省略，尽量不要省略,因为有时会有歧义的情况
			JavaScript的执行顺序：按照HTML文件中出现的顺序依次执行
			JavaScript忽略空白符和换行符
			JavaScript通过\ 对代码进行折行操作

标识符命名和注释

			firstSecond   myCar	doSomethingImportant
			注意：构造函数的首字母一般使用大写字母开头。
			1. 只能用数字、字母、下划线、$
			2. 不能使用数字开头
			3. 区分大小写
			4. 不能用关键字
			5. 驼峰命名法 第一个字母小写，剩下的每个单词的首字母大写
			6. 单行注释。	//这里是注释，只能写一行
			7. 多行注释。    /* 这里的注释可以写多行  */
				快捷键：单行：ctrl+/；多行：/**+回车

变量声明

			在js中定义的全局变量都会成为window对象的属性
			下面定义的的变量num，相当于window.num=10
			var num = 10//=不叫等于号，而称为赋值；数值符号数值==，字符===
			因为对象属性是惟一的，不会重复，故变量名重复不会报错
			同一个变量，可以声明多次，而且后面的声明也不会让前面已经赋的值丢失，但是赋值就会
			建议一行定义一个变量，定义变量最好一块定义
			但是即使不放一起也会自动提前，因为都是放windows对象上的，windows是全局的，所以会提前执行变量定义
