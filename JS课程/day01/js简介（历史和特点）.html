js是什么

			JS是弱类型语言,没有类型声明,它的变量不必具有一个明确的类型
			JS是脚本语言，以用来编程的并且直接执行源代码的语言
				拓展：机器语言->汇编语言->高级语言
				各种动态语言，如ASP、PHP、CGI、JSP等，都是脚本语言
				脚本语言相比编译型语言，优点就是轻量级，无需编译，它们依赖的平台就是其
				解释执行器，Html依赖html解释执行器，javascript依赖js解释执行器
			JS也是解释性的语言，在运行的时候将程序直接翻译成机器的语言
			基于对象(Object)和事件驱动（如按钮点击，触摸屏）(Event Driven)
			并具有安全性能的脚本语言

js历史

			1990 欧洲核能研究组织（CERN）科学家Tim Berners-Lee（蒂姆 伯纳斯-李），互联网之父
			1992 美国国家超级电脑应用中心（NCSA）开发独立浏览器Mosaic （马赛克）
			1994年10月 NCSA的一个程序员成立Mosaic通信公司，后改名网景Netscape
			1994年12月 面向普通用户的新一代的浏览器 Netscape Navigator1.0
			1995年5月，Netscape公司Brendan Eich只用了10天，就设计完成了这种语言的第一版liveScript。
			1995年12月4日，Netscape公司与Sun公司联合发布了JavaScript语言（因是当时Java语言非常红火，所以网景公司希望借Java的名气来推广，但事实上JavaScript除了语法上有点像Java，其他部分基本上没啥关系）
			1996年8月 微软模仿JavaScript开发了一种相近的语言，取名为JScript，Netscape公司面临丧失浏览器脚本语言的主导权的局面
			1997 网景公司将javaScript交给ECMA(European Computer Manufacturers Association)组织，以此来抵制微软的垄断
			2011年ECMAscript 5.1版发布，并且成为ISO国际标准（ISO/IEC 16262:2011）。到了2012年底，所有主要浏览器都支持ECMAScript 5.1版的全部功能
			2015年 ECMAScript6.0 改名为 ECMAScript2015
			每年发布一个所以用年数来命名ECMAScript2016

特点

简单性（弱语言类型）、安全性（无法访问本地硬盘？a的download也不行？）、 动态性（可以直接对用户或客户输入做出响应，无须经过Web程序）、 跨平台性（JS依赖于浏览器本身，与操作环境无关）

理论体系：ECMAScript（核心）+BOM(broswer object model 浏览器对象模型)+DOM(document object model 文档对象模型)
