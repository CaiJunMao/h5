运算符

			基本理解：
				运算符也叫操作符
				在应用于对象时，相应的操作符通常都会调用对象的valueOf()和（或） toString() 方法，以便取得可以操作的值
			加号运算符
				 * infinity+infinity->NaN，而不是为0
				 * 如果有一个是字符串，就按照字符串的串联的方式连接。
				 * (如果另外一个不是字符串，则先转换成字符串再连)
				 * 可以用来直接连接字符串
			减法/除法/乘法
				
				 * 如果不全是数字，运算之前先调用Number进行转化，再进行运算
				 * 有对象参与时：先调用valueOf(),没有valueOf()则调用toString()
				 除数为0不报错  ，0/0不是无限大而是NaN
			求余运算 %
			一元运算符 -- ++
				 * a=10,对b=++a和b=a++;前者b=11，后者b=10,
				 * 因为前者a是先自增载赋值给b
				var m = 1;
				console.log(m++ + ++m);//1+3=4
			赋值运算符=和复合赋值运算符+=、-=、*=、/=、%=
				var n = 2;
				n *= 4 + 3//是等价于先算4+3还是n*4+n*3还是n*4+3
				//优先级低于+-*/所以为前者，结果为14
			比较运算符 ==/!= 和===/!==
				 * 
				 * 	有Number先都转化为Number
				 * 	都是字符串 比较的是字符串中的字符在字母表中的顺序。(或者说比较是他们的编码值)
				 * 	如果有一个是NaN，则结果一定是false。==但是有一种情况，如果比较特殊NaN  !=  NaN则是true
				 null == unfinded，但是null !== undefined
				 * == 判断两个操作数值
					 * 两个对象比较 比较两个对象是否为同一个对象。是就相同，不是就不同
					 * 一个对象和一个基本类型比较，则调用对象的valueOf方法，转变成基本类型之后再比较
				 * === 全等判断，判断类型和值
				 * 不做任何的转换 
			逻辑运算符 && || ！
				 * （面试题）都是短路形式：如&&，有个为假后一个不用判断了（因为要全为真），直接短路
			三元运算符（面试题）
				 * 逻辑值（表达式） ？ 表达式1：表达式2；
				 * 如果逻辑值不是Boolean值，先转型
			typeOf操作符 控制台显示时类型为小写
				 * null:object
				 * Number:number
				 * 函数：function
				 * 函数（）：返回值的数据类型?
				 * NaN:numbeer
				 * arr：object
			逗号运算符
				 * 返回最后一个值
				 * 使用逗号操作符可以在一条语句中执行多个操作
