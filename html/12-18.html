1 元素

HTML 元素指的是从开始标签（start tag）到结束标签（end tag）的所有代码，例如p h1这些都是元素。大多数元素都是一对的，需要开始和闭合，空元素（没有内容的元素）在开始标签中进行关闭
元素的内容是开始标签与结束标签之间的内容
，是关闭空元素的正确方法

2 属性

HTML 标签可以拥有属性。属性提供了有关 HTML 元素的更多的信息。
属性总是以名称/值对的形式出现，比如：name="value"
属性总是在 HTML 元素的开始标签中规定
HTML 提示：使用小写属性;始终为属性值加引号,单双引号都ok,已有双引号则用单引号，如：name='Bill "HelloWorld" Gates'

4 style属性

提供了一种改变所有 HTML 元素的样式的通用方法；HTML 4 引入的，它是一种新的首选的改变 HTML 元素样式的方式。通过 HTML 样式，能够通过使用 style 属性直接将样式添加到 HTML 元素，或者间接地在独立的样式表中（CSS 文件）进行定义

下面的标签和属性将被放弃，请尽量使用style代替： center 定义居中的内容。
font 和 basefont 定义 HTML 字体。
s 和 strike 定义删除线文本
u 定义下划线文本
属性 描述
align 定义文本的对齐方式
bgcolor 定义背景颜色
color 定义文本颜色

用下列style属性代替：background-color，font-family、color 以及 font-size，text-align
